꿈비 안성창고 재고 혼선 방지 툴

B2B 배송대기(할당) 실시간 체크 및 로케이션 이적 검증 시스템 (GitHub Pages)

Step 1: 당일 WMS 재고 데이터 입력 ● 데이터 없음 (대기 중)
📊

WMS에서 내보낸 CSV 파일을 여기에 드래그하거나 클릭하여 업로드하세요

(엑셀에서 '다른 이름으로 저장' -> 'CSV (쉼표로 분리)(*.csv)' 파일 선택)

WMS 엑셀을 열고 전체 데이터(헤더행 포함)를 드래그해서 복사(Ctrl+C)한 후 아래에 붙여넣기(Ctrl+V) 하세요.

Step 2: 현장 이동 작업 검증 (PDA / 모바일 지원)

현장에서 박스를 이동시키기 전, 로케이션 바코드 또는 상품 바코드를 스캔해 보세요.

Step 3: 이동 금지 / 주의 관리 대상 모니터링
상품명 바코드 창고 다중로케이션 재고수량 배송대기 가용재고 상태
WMS 데이터를 업로드하거나 데모 데이터를 불러오면 목록이 표시됩니다.
💡 엑셀 복사/붙여넣기 및 활용 가이드
  • 초간단 복사/붙여넣기 지원: 엑셀 파일을 일일이 다른 이름으로 저장할 필요 없이, 엑셀의 데이터 영역을 마우스로 쭉 긁어서 복사한 다음 '엑셀 복사해서 붙여넣기' 탭에 붙여넣기만 해도 완벽히 작동합니다!
  • 자동 컬럼 매핑 탑재: 업로드하신 파일의 첫 번째 줄(제목)을 분석하여 아래의 순서와 이름을 자동으로 맞춰줍니다.
    상품명 | 바코드 | 창고 | 다중로케이션 | 재고수량 | 배송대기 | 가용재고
  • 로컬 보안 보장: 모든 데이터 처리는 밖으로 나가지 않고 사용자의 PC 내부에서만 실행되어 정보 유출 우려가 전혀 없습니다.
