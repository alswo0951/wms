꿈비 안성창고 재고 혼선 방지 툴

B2B 배송대기(할당) 실시간 체크 및 로케이션 이적 검증 시스템 (GitHub Pages)

Step 1: 당일 WMS 재고 데이터 업로드 (CSV) ● 데이터 없음 (대기 중)
📊

WMS에서 내보낸 Excel/CSV 파일을 여기에 드래그하거나 클릭하여 업로드하세요

(형식: 상품명, 바코드, 다중로케이션, 재고수량, 배송대기, 가용재고 필드 포함)

Step 2: 현장 이동 작업 검증 (PDA / 모바일 지원)

현장에서 박스를 이동시키기 전, 로케이션 바코드 또는 상품 바코드를 스캔해 보세요.

Step 3: 이동 금지 / 주의 관리 대상 모니터링
상품명 바코드 로케이션 재고수량 배송대기 가용재고 상태
WMS 데이터를 업로드하거나 데모 데이터를 불러오면 목록이 표시됩니다.
💡 깃허브 페이지를 통한 현장 배포 가이드
  • 동작 원리: 서버가 없이 사용자의 브라우저 메모리 안에서 데이터가 즉시 처리되므로 개인정보가 안전하게 보호됩니다.
  • 매일 아침 업데이트 루틴:
    1. 출근 후 WMS에서 [현재고 현황 조회]를 실행합니다.
    2. 조회된 화면을 Excel 또는 CSV 파일로 다운로드합니다.
    3. 이 깃허브 페이지에 접속하여 다운받은 파일을 드래그 앤 드롭으로 넣습니다.
    4. 현장 작업자들이 각자의 스마트폰/PDA로 이 페이지 주소에 접속하여 검증 기능을 사용합니다.
